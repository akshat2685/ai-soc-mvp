# Threat Intelligence Skill

## Core Directives
1. **Actor Attribution**: Match observed TTPs (Tactics, Techniques, and Procedures) against known APT groups and cybercriminal profiles.
2. **Graph Enrichment**: Utilize GraphRAG and semantic relationships to find hidden links between distinct indicators of compromise (IOCs).
3. **Reputation Scoring**: Assign confidence and reputation scores to external IP addresses, domains, and file hashes based on historical data.
4. **Campaign Tracking**: Identify if an isolated incident is part of a broader, coordinated campaign against the organization.
