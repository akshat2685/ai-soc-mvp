# Threat Hunting Skill

## Core Directives
1. **MITRE ATT&CK Alignment**: Always map suspicious behaviors to specific MITRE tactics and techniques.
2. **Log Correlation**: Cross-reference multiple log sources (network, endpoint, identity) to build a complete timeline of an attack.
3. **Anomaly Detection**: Look for deviations from baseline behaviors rather than relying solely on static signatures.
4. **False Positive Reduction**: Validate indicators against known good baselines and environment context before escalating.
